## Extension Settings
In settings.json provide the link to your proejct
```json
  "issueTracker.gitLabProjectURL": "https://gitlab.com/???/???/???"
```
Specify which files to ignore
 ```json
    "issueTracker.foldersToIgnore":[
        "./.git",
        "./coverage",
        "./node_modules",
        "./src/downloads",
        "./src/uploads",
    ]
```
